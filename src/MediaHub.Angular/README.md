# MediaHub Angular Scaffold

This is a lightweight folder scaffold meant to be copied into your Angular workspace.

## Suggested destination
- `apps/cc-client/src/app/media-hub/` (or similar)

## Contains
- Models (DTOs that match the API)
- Services (polling feed client + admin clients)
- Components:
  - `media-headlines-list` (compact list)
  - `media-headlines-ticker` (ticker strip)
