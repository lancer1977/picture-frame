# Personas and Use Cases

## Personas
### Streamer (Primary)
- Wants quick headlines and a “what’s new” scan
- Needs a display that looks good on stream and doesn’t distract

### Curator/Admin (Secondary)
- Manages subscriptions and rules for a channel
- Hides noisy sources, features certain creators

### Plugin developer (Future)
- Implements a provider in another repo/service (Twitch plugin, etc.)
- Needs stable interfaces + DTOs to plug in

## Use cases
- Add a YouTube channel subscription via RSS/ID
- Add a podcast via RSS URL
- See latest items with thumbnails and age
- Mark item as hidden/saved/featured
- Output a “headline strip” feed for OBS browser source
