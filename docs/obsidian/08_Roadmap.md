# Roadmap

## v0 (spike)
- YouTube RSS provider (channel uploads feed)
- Podcast RSS provider
- Basic storage + feed endpoints
- Simple stream list component in Angular

## v1 (usable)
- Per-channel subscription management UI
- Curation states (hide/save/feature)
- ETag support + efficient polling
- Browser-source mode UI

## v2 (nice)
- Twitch clips/VOD metadata (via plugin layer)
- Tagging and rules (e.g., duration filters)
- Shared “topic lists” (Tech, DevOps, Security)
- Optional caching of thumbnails (CDN) to avoid cross-site issues
